In der nachfolgenden Visualisierung wird ein Balkendiagramm gezeigt. Um mit Balkendiagrammen zu interagieren, navigieren Sie zu der Visualisierung und drücken Sie anschließend Enter. Danach gelangen Sie zu einem Interaktionsmenü, das Ihnen mehrere Interaktionsmöglichkeiten auflistet. Die einzelnen Interaktionsmöglichkeiten können Sie mit Hilfe von Enter aktivieren. Im nachfolgenden Menü werden die einzelnen Interaktionsmöglichkeiten erläutert.
  • Navigieren durch die Daten
    • Der Fokus wird direkt auf den ersten Datenpunkt in der Visualisierung gesetzt. Sie können mit den Pfeiltasten Links und Rechts durch die Daten navigieren.
    • Wenn Sie den Fokus auf einem Datenpunkt haben können Sie den Datenpunkt mit Hilfe von SHIFT+M makieren. Dieser Datenpunkt wird daraufhin in einer Liste gespeichert, nach der Sie anschließend in einem seperaten Interaktionsmenü filtern können.
    • Drücken Sie SHIFT+S während sich der Fokus auf einem Datenpunkt befindet, wird die auditive Repräsentation dieses Datenpunktes abgespielt.
    • Drücken Sie ESCAPE um wieder zurück zum Interaktionsmenü zu gelangen.
  • Zusammenfassung
    • Es wird eine Zusammenfassung über die Visualisierung ausgegeben: was für Daten werden gezeigt, welche Daten werden von den einzelnen Achsen repräsentiert, wie ist der Datenverlauf, usw.
  • Suchen
    • Es öffnet sich ein Untermenü wo Sie auswählen können, ob Sie nach einem Wert auf der y-Achse oder einem Datenpunkt-Label suchen wollen
    • Bei der Suche nach einem y-Wert können Sie die Symbole < und > verwenden oder nach einem exakten Wert suchen
    • Bei der Suche nach einem Datenpunkt-Label verwenden Sie statt Umlauten bitte die ausgeschriebene Variante davon - statt ä schreiben Sie bitte ae
  • Sonification
    • Die Daten werden in einer auditiven Darstellung wiedergegeben, die es Ihnen ermöglicht, den Verlauf der Daten akustisch wahrzunehmen.
  • Makierungen
    • Es öffnet sich ein weiteres Untermenü, in dem Sie auswählen können, ob Sie die aktuell gesetzten Makierungen löschen wollen oder ob Sie die abgebildeten Daten nach den makierten Datenpunkten filtern wollen.
  • Daten zurücksetzen
    • Die Visualisierung wird auf den Ausgangszustand zurückgesetzt - alle Filterungen werden aufgehoben, doch makierte Datenpunkte bleiben erhalten.
In der nachfolgenden Visualisierung wird ein Liniendiagramm gezeigt. Um mit Liniendiagrammen zu interagieren, navigieren Sie zu der Visualisierung und drücken Sie anschließend Enter. Danach gelangen Sie zu einem Interaktionsmenü, das Ihnen mehrere Interaktionsmöglichkeiten auflistet. Die einzelnen Interaktionsmöglichkeiten können Sie mit Hilfe von Enter aktivieren. Im nachfolgenden Menü werden die einzelnen Interaktionsmöglichkeiten erläutert.
  • Navigieren durch die Daten
    • Der Fokus wird direkt auf die erste Linie in der Visualisierung gesetzt. Sie können mit den Pfeiltasten nach unten und oben zu den anderen Linien navigieren.
    • Steht der Fokus auf einer Linie, kann mit Hilfe von ESCAPE zum Interaktionsmenü navigiert werden. Mit Hilfe von SHIFT+M können alle Datenpunkte innerhalb der Linie makiert werden, SHIFT+S gibt eine auditive Repräsentation der Linie wider.
    • Mit Enter navigieren Sie in die fokussierte Linie hinein, wobei der Fokus auf den ersten Datenpunkt der Linie gesetzt wird. Danach können Sie mit den Pfeiltasten nach links und rechts zu den übrigen Datenpunkten navigieren.
    • Befindet sich der Fokus auf einem Datenpunkt innerhalb einer Linie, kann mit Hilfe von ESCAPE zur Linie selbst zurück navigiert werden.
    • Wenn Sie den Fokus auf einem Datenpunkt haben können Sie den Datenpunkt mit Hilfe von SHIFT+M makieren. Dieser Datenpunkt wird daraufhin in einer Liste gespeichert, die Sie anschließend filtern können.
    • Beim Fokus auf einem Datenpunkt können Sie ebenfalls SHIFT+S drücken um die auditive Repräsentierung des Datenpunktes abzupielen.
  • Zusammenfassung
    • Es wird eine Zusammenfassung über die Visualisierung ausgegeben: was für Daten werden gezeigt, welche Daten werden von den einzelnen Achsen repräsentiert, wie ist der Datenverlauf, usw.
  • Suchen
    • Es öffnet sich ein Untermenü wo Sie auswählen können, ob Sie nach einem Datum oder einem Linien-Label suchen wollen
    • Bei der Suche nach einem Datum können Sie die Symbole < und > verwenden oder Sie können nach einem exakten Wert suchen
    • Bei der Suche nach einem Linien-Label verwenden Sie statt Umlauten bitte die ausgeschriebene Variante davon - statt ä schreiben Sie bitte ae
  • Sonification
    • Die Daten werden in einer auditiven Darstellung wiedergegeben, die es Ihnen ermöglicht, den Verlauf der Daten akustisch wahrzunehmen.
  • Makierungen
    • Es öffnet sich ein weiteres Untermenü, in dem Sie auswählen können, ob Sie die aktuell gesetzten Makierungen löschen wollen oder ob Sie die abgebildeten Daten nach den makierten Datenpunkten filtern wollen.
  • Daten zurücksetzen
    • Die Visualisierung wird auf den Ausgangszustand zurückgesetzt - alle Filterungen werden aufgehoben, doch makierte Datenpunkte bleiben erhalten.

    • Befindet sich der Fokus auf einer Gruppe von Datenpunkten, können Sie mit Hilfe von ENTER in diese Gruppe navigieren, wobei der Fokus auf den ersten Datenpunkt in dieser Gruppe gesetzt wird.
    • Befindet sich der Fokus auf einem Datenpunkt innerhalb einer Gruppe, so gelangen Sie mit ESCAPE zurück zu der Gruppe selbst.

    • Bei der Suche nach einem Datenpunkt-Namen verwenden Sie statt Umlauten bitte die ausgeschriebene Variante davon - statt ä schreiben Sie bitte ae
